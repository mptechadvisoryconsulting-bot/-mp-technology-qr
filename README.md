# MP Technology QR

Branded QR code generator for MP Technology Consulting.

## Local Web Build

```powershell
npm run build:web
```

The static site is generated into `public/`.

## Vercel

This repo is ready for Vercel:

- Build command: `npm run build:web`
- Output directory: `public`

Vercel reads these settings from `vercel.json`.

## Desktop Build

```powershell
npm run build:win
```

The packaged Windows app is generated in `dist/`.
